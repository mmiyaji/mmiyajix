  2. アップロード

    上でも言ったよね？次いくよ

  3. ユーザー登録

    ユーザは複数登録可能であるが，ユーザにはグループがあり，そのグループごとに権限が異なります．

    Admin :
    アプリケーションの管理者．あらゆる実行権限を持つ．
    Developper :
    レイアウトの変更，バージョン管理，記事執筆の権限を持つ．
    Writer :
    記事執筆権限を持つ．
    User :
    通常のユーザ．外部アプリケーションとの連携やAPIを利用するためのTokenを発行可能．
    まずはAdminユーザを作成し，アプリケーションの設定を行う必要があります． Create User

  4. アプリケーション登録

    このサイトのタイトルやレイアウトの初期設定を行います．この操作にはAdminグループに属するユーザとしてログインしている必要があります． Initialize Applicaion
